# Мой Трекер Целей

## Особенности

- Создание своих целей – добавляйте любые темы для изучения
- Визуальный прогресс – цветные карточки и прогресс-бар
- Персональные заметки – сохраняйте идеи, конспекты, ссылки
- Планирование сроков – устанавливайте дедлайны
- Импорт/Экспорт – работа с JSON файлами
- Автосохранение – все данные в LocalStorage
- Адаптивный дизайн – работает на телефонах и компьютерах
- Современный интерфейс – иконки, градиенты, тени
